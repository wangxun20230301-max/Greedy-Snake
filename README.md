# 🐍 贪吃蛇大冒险

一个专为6-12岁儿童设计的经典贪吃蛇游戏，界面简洁有趣，寓教于乐！

## ✨ 游戏特色

- 🎨 **儿童友好界面** - 鲜艳的色彩、可爱的动画和表情符号
- 🎯 **三种难度级别** - 简单、中等、困难，适合不同年龄段
- 🏆 **本地最高分记录** - 自动保存最佳成绩
- ⭐ **特殊奖励食物** - 星星食物提供额外加分
- 📱 **响应式设计** - 完美支持桌面和移动设备
- ⏸️ **暂停/继续功能** - 随时暂停游戏
- 🎮 **双重控制方式** - 键盘方向键 + 移动端触控按钮

## 🚀 快速开始

### 在线体验
直接在浏览器中打开游戏即可开始游玩！

### 本地运行
1. 克隆或下载项目
2. 使用任意本地服务器启动，例如：

```bash
# 使用 Python
python3 -m http.server 8000

# 使用 Node.js (需要先安装 http-server)
npx http-server
```

3. 在浏览器中访问 `http://localhost:8000`

## 🎮 游戏说明

### 操作方式
- **桌面端**：使用键盘方向键 ↑ ↓ ← → 控制蛇的方向
- **移动端**：使用屏幕下方的方向按钮控制
- **暂停/继续**：点击暂停按钮或按空格键

### 游戏规则
- 🍎 吃到苹果获得 **10分**，蛇身变长
- ⭐ 吃到星星获得 **50分** 特殊奖励（星星会在5秒后消失）
- 💥 撞到墙壁或自己的身体则游戏结束
- 🎯 难度越高，蛇移动速度越快，得分挑战越大

## 🛠️ 技术栈

- **CSS3** - 渐变背景、动画效果和响应式设计
